#  HoyStore – Loja de Hardware

##  Sobre o Projeto

Este projeto foi desenvolvido para a disciplina de **Programação Web** na **UFC - Campus Itapajé**, ministrada pelo professor **Aldisio Gonçalves Medeiros**.  
O **HoyStore** é um projeto de uma aplicação web simples desenvolvida para fins acadêmicos, contendo:

- Página inicial com categorias e produtos

- Página de detalhes de produto

- Página de cadastro de usuário

- Página de carrinho de compras

O objetivo principal é demonstrar o uso de HTML semântico, manipulação básica com JavaScript e organização de páginas e componentes visuais com CSS.

---
##  Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**

---

##  Funcionalidades


### Página Inicial

- Exibe categorias e produtos
- Cards com imagem, nome e destaque visual

### Página de Detalhes

- Informações completas do produto
- Imagem destacada
- Lista de especificações

###  Formulário de Cadastro

- Campos personalizados
- Validação em JavaScript (CPF, email, senha…)

###  Carrinho de Compras

- Exibe lista de itens com imagem
- Campo de quantidade modificável
- Alerta ao tentar colocar “0” itens
- Botão de finalizar compra

---

###  Como execultar o projeto

1. Baixe ou clone o repositório:
   ```bash
   https://github.com/hoyalles/HoyStore.git
   ```
2. Abra o projeto em qualquer navegador:
-  Dê duplo clique no home.html
- Ou utilize o Live Server do VS Code
  
Não é necessário instalar nenhuma biblioteca ou dependência, o projeto é totalmente estático.
